# 文件及接口命名规范
1、文件保持剪短易懂，保持小驼峰规则：demoUser \
2、前端及数据接口对应规则：

| 接口   | 前端     | 后端         |
|------|--------|------------|
| 读取   | search | read/get   |
| 写入   | add    | create/set |
| 编辑   | edit   | update     |
| 删除   | remove | delete     |
