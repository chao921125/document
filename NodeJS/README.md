###### nodejs 支持 ESM 方式
```text
1.将文件后缀改为.mjs
2.在项目中package.json新增配置项"type":"module"，那么整个项目中的.js文件都会按照ESM规范去执行
3.增加执行参数--input-type也可以实现相同效果
```
